# Pizza Sales Performance Analysis

This project is my hands-on practice in turning raw sales data into a clear business performance story.

Using the Pizza Sales dataset, I focused on a few questions that matter in real business settings:
How much revenue are we generating and what is the average order value?
When are our busiest days and peak hours?
Which pizza categories and sizes drive the most sales?
And what are our top 5 best-selling vs. bottom 5 worst-selling pizzas?

To answer those questions, I used SQL to prepare and analyze the data, then built an Excel dashboard to present the results in a business-friendly way.

## What this project covers
- High-level KPIs calculation (Revenue, Average Order Value, Total Orders)
- Daily and hourly order trend analysis
- Sales distribution by pizza category and size
- Top 5 best and bottom 5 worst performers breakdown
- Interactive dashboard in Excel

## Tools
- MS SQL Server
- MS Excel

## Dataset
- Pizza Sales Dataset
- Main table used: `pizza_sales`

## Project structure
- `data/` contains the raw dataset file used in the project
- `sql_queries/` contains the SQL scripts for data extraction
- `dashboard/` contains the final interactive Excel workbook
